Lab 1

Notepad - a simple plaintext editor for Microsoft Windows and a basic text-editing program which enables you to create documents. Does not have any formatting capabilities.

WordPad - a simple document editor. It has the ability to use formatting (fonts, bolding, italics, etc.) and control layout (page size, justification, margins, etc.). However, it lacks functions such as spell checker, thesaurus, control over pagination, etc….

Microsoft Word - a widely used commercial word processor designed by Microsoft. Microsoft Word is a component of the Microsoft Office suite of productivity software but can also be purchased as a stand-alone product.
